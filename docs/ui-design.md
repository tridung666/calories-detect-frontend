# UI Design

Goal:

Modern SaaS Dashboard

---

## Design Principles

- Clean
- Minimal
- Professional
- Spacious
- Responsive

---

## Layout

Desktop first.

Support Mobile.

Maximum content width:

1280px

---

## Border Radius

Prefer rounded-lg

Avoid sharp corners.

---

## Shadows

Soft shadows only.

Avoid heavy shadows.

---

## Typography

Use clear hierarchy.

Heading

↓

Subheading

↓

Body

↓

Caption

---

## Colors

Neutral background.

Primary color for actions.

Red only for destructive actions.

Green only for success.

---

## Buttons

Primary

Secondary

Outline

Ghost

Destructive

Follow shadcn conventions.

---

## Cards

Use cards for:

Dashboard

Statistics

Meals

Foods

Profile

---

## Spacing

Prefer:

4

8

12

16

24

32

48

Avoid random spacing values.

---

## Icons

Use Lucide Icons.

Keep icon sizes consistent.

---

## Tables

Support:

- Empty state
- Loading state
- Pagination

---

## Forms

Always include:

- Labels
- Validation
- Error messages

---

## Accessibility

Use semantic HTML.

Support keyboard navigation.

Add aria-label when appropriate.

---

## Responsive

Support:

Mobile

Tablet

Desktop

Never hardcode widths.

Prefer Flexbox and Grid.
